# Java Reference

## Prerequisites
- **Java Development Kit (JDK)** 8 or higher installed.
- A terminal or command line interface.

## How to Compile

To compile the Java source file, navigate to this directory in your terminal and use the `javac` compiler:

```bash
cd 217. Contains Duplicate
javac ContainsDuplicate.java
```

This will compile the code and generate a `ContainsDuplicate.class` bytecode file in the same directory.

## How to Run

After successfully compiling, you can run the program. The `ContainsDuplicate` class contains a `main` method with multiple test cases covering standard inputs and edge cases. Run it using the `java` command:

```bash
java ContainsDuplicate
```
